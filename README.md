# Laptop-Market-Intelligence-Dashboard


## verview

This project analyzes over **8,000 laptop models** to provide insights into market trends, brand performance, hardware specifications, and pricing using **Power BI**.

The dashboard is designed to help users compare laptop brands, understand market segmentation, and explore key hardware features

## Dashboard Features

### Executive Overview
- Total Laptop Models
- Average Price
- Highest Price
- Average Battery Life
- Average Refresh Rate

### Brand Analysis
- Top 10 Laptop Brands
- Models by Brand
- Average Price by Brand
- Average RAM by Brand
- Average SSD Capacity by Brand

### Market Analysis
- Processor Brand Distribution
- Price Segment Distribution
- Operating System Analysis
- Interactive Filters

## Key Insights

- Compared the top laptop brands based on pricing and specifications.
- Analyzed processor market share across major brands.
- Identified laptop distribution across different price segments.
- Evaluated RAM, SSD, battery life, and display specifications.

## Tools Used

- Power BI

- DAX

Real-world laptop specification dataset containing over **8,000 laptop models** with information on:
